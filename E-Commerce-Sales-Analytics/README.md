## 📌 Project Description
This project simulates a real-world e-commerce sales analytics pipeline. It includes:
- **Synthetic data generation** with realistic patterns, missing values, duplicates, and outliers.
- **Data cleaning and feature engineering** using Python (Pandas).
- **Exploratory Data Analysis (EDA)** with visualizations and statistical insights.
- **Interactive Power BI dashboard** for business intelligence.
- **Actionable business insights** derived from the data.

The project demonstrates proficiency in data engineering, analysis, and visualization – suitable for a Senior Data Analyst/Data Engineer role.

## 🎯 Objectives
- Analyze customer purchasing behavior and seasonal sales trends.
- Identify top-performing products, categories, and regions.
- Evaluate customer segments and payment method performance.
- Provide recommendations for marketing, inventory, and logistics.

## 📂 Dataset
- **Source:** Synthetic data generated using Python (15,000+ rows).
- **Features:** Order details, customer demographics, product attributes, financials, delivery status, ratings.
- **Cleaned version:** `data/processed/cleaned_sales.csv`.

## 🛠️ Technologies Used
- **Python 3.8+** – Pandas, NumPy, Matplotlib, Seaborn, Plotly, Scikit-learn.
- **Jupyter Notebook** – Interactive EDA.
- **Power BI Desktop** – Dashboard creation.
- **Git & GitHub** – Version control.
